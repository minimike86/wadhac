NIST Cybersecurity Framework

The Cybersecurity Framework was created in response to Executive Order 13636, which aims to improve the security of the nation's critical infrastructure from cyber attacks.
While directed to "critical infrastructure" organizations, the Framework is a useful guide to any organization looking to improve their cyber security posture. The document is divided into the framework core, the implementation tiers, and the framework profile. The framework core describes 5 functions of an information security program: identify, protect, detect, respond, and recover. The Framework references other documents like NIST 800-53 and COBIT 5 for specific controls and processes needed to implement these functions.
